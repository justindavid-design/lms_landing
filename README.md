# LMS Landing (React + MUI + Tailwind)

This is a minimal starter for the landing page based on the provided design.

Quick start

1. Install dependencies:

```bash
cd /workspace/lms-landing
npm install
```

2. Run dev server:

```bash
npm run dev
```

Notes
- Uses Vite for fast dev server
- Tailwind is configured in `tailwind.config.cjs`
- Material UI components are used for buttons and layout bits
